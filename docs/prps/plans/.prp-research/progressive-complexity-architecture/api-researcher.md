# api-researcher — Progressive Complexity Architecture (#29)

**Verdict: No external research needed for this feature — internal only.** No external HTTP APIs, SDKs, or new runtime dependencies are warranted. The automatic-progression heuristic and tier-context are covered by existing Svelte stores + Svelte's built-in context API. The relevant "contract" to mirror is the internal contract-first flow (OpenAPI → `openapi-typescript` → `v1.d.ts` → `+server.ts` + raw-`fetch` client store). Documented below.
